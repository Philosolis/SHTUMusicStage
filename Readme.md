# 简介
（这里应该有图，但小舞台海报太大了，影响观感，就先不放了）

**小舞台**是由上海科技大学音乐社每周举办的活动，旨在为所有想要展现自我的人提供一个轻松的舞台。这里对所有音乐，无论其艺术形式和风格，都一视同仁地接纳；对所有希望上台表演的人，都给予真诚的鼓励，毕竟万事开头难。

小舞台的主要负责人每周组织，召集志愿者搭建舞台、调音、摄影、主持，邀请观众和志愿者们上台演唱。理论上，负责人在音乐社内部轮替交接，但只要你想，你也可以成为小舞台之主。当然，负责人本身也是志愿者的一员，组织者需要比一般志愿者付出多一点，就一点。

小舞台没有强制出勤、考核或是职务安排，因为用不着。大多数时候，3-5名志愿者可以支撑一次完整的活动，如要搬运设备到室外场地，可能需要稍多人手，6-8名就足矣。

以下我会撰写一篇指南，教会你如何高效搭建一个舞台，如何理解“舞台”和调音系统，以及小舞台是如何运作。

# 指南
音乐类活动不分大小，其内在逻辑是一致的。从大规模音乐节到Livehouse的演出，你都需要理解以下内容：

## 舞台
舞台就是演出者表演的地方，以小舞台为例，每次志愿者们需要布置调音台、音响、乐器、各类架子以及灯光。

### 调音台
不要因这名词而生畏，调音台的逻辑其实很简单：将声音信号通过不同的线缆集中到声卡，通过电脑操纵声卡，你就能整理声音信号，把他们合到一起，再发送到该去的地方。

目前，小舞台使用志愿者的电脑，装载宿主程序Studio One以及雅马哈声卡驱动，操纵一台雅马哈声卡来实现大多数调音功能。

#### 声卡
不同的声音信号通过线缆接入到声卡，这是第一步。下图展示了小舞台目前使用的雅马哈声卡，它有8组输入和输出，同时兼容6.35mm音频线和卡农线。

##### 6.35mm音频线
[6.35mm线图]
两端都是6.35mm的线缆同时也是吉他最常用的线缆，有时候似乎会被简略称作6.5线，你只需要知道吉他线、6.5线指的都是这种线缆即可。在小舞台，它常用于链接吉他、合成器（电钢琴）到调音台/声卡。

> **有点用的小知识**
> 6.35mm音频线事实上有平衡和非平衡的区别，简单来说，非平衡信号只有接地和音频本身的信号，用音频信号减去接地（0信号）就是信号的数值。而非平衡信号会将音频信号的反相信号[^3]再发送一次，最后用正信号减去负信号再除2就得到了音频信号。
> 
> 这样做有什么好处呢？假如一个外界干扰对线缆的两个信号产生相同的影响，那么它在正信号上的影响和在负信号上的，经过这一处理就会刚好相互抵消。也就是说，平衡线的抗干扰能力更强，更适合长距离传输。
> 
> 吉他线通常是非平衡的，有一部分原因是，电吉他是一个相当有年头的乐器，设计之初就没有做平衡信号输出的传统（要额外花钱的啦）。不过数字吉他效果器通常都可以转换输出平衡信号。更大规模的舞台会使用DI盒这种设备，它的一个重要功能就是将非平衡信号转化为平衡信号再发送。当然，DI盒还有别的功能，这里先不赘述。
> 
> 如果你想要辨认一根6.35mm线是平衡还是非平衡，可以看它上面的黑色圆环数量，那就是它能传输的不同信号的数量。如上所述，两个环最多只有接地和音频信号，那当然是非平衡线，而三个环一般就可以传输平衡信号。当然，它的两路输出也可以传输左右声道的信号，具体问题具体分析。
> 
> 不过，你完全不需要过度在意这件事，大多数平衡信号输入/输出都兼容非平衡的，混用通常不会导致什么灾难性事故。
> 
> 我说的这些不一定对，更严谨的定义请自行查阅相关资料。

##### 卡农线
[XLR线图]
这种两端不一的线材俗称卡农线，事实上正经叫做XLR线（XLR cable）。这是麦克风的通用线材，你可以从它两端的形制不同很轻易地猜到哪里该接麦克风，哪里该接调音台。注意，它无论哪一端都只能通过特定角度接入，不要大力出奇迹！

> **没什么用的小知识**
> 据说这种线叫卡农是因为国内最初是Canon（佳能）在推广，和音乐中的卡农一点关系都没有。但这是GPT说的，最好有人查证一下。

#### 电脑
宿主是负责调度音频流的软件，声音信号接入声卡，通过宿主来操纵它，对声音进行各种处理，最终电脑处理后的声音信号通过声卡又输出左右声道分别到两个音响。

#### 音响
小舞台使用两台Joyo AC20音响，便宜耐用，通常由电脑输出的音频信号通过两根6.35mm/3.5mm音频线[^1]，各接一根**特制线缆**来修复音响的声音反相导致的音量问题，其中一端需要额外接入**屏蔽器**来消除电流噪音。

##### 特制线缆
这是张闻朔同学（不要把他名字里的朔打错了，是朔不是硕）根据问题原因手动焊接的两根线缆，在此前音箱的左右音量似乎由于反相相消会不均衡，具体我记不太清，得问问他。如果他有空，我会喊他在这里补充一大串理论知识。

小舞台的志愿者需要注意，这两根手作线缆十分脆弱，一定不要拉伸、弯折，在接入设备时将他们连接的6.35mm线缆在音响底部缠绕几圈作缓冲最佳，这样一旦有牵扯会是缠绕部分承受拉力。[^2]

[吉他手将线绕过背带的图]

##### 屏蔽器
我同样不知道具体的原理，但是，一定要在一端的输出线缆连接屏蔽器后再接入音响，否则噪音够你受的。
